# Avatar Index - Store + App Launch Kit

9 distinct buyer avatars for targeted messaging in **offer building**.

**Important**: These are Product Avatars (target customer profiles) - different from the 5 Audit Personas in `04_CREATIVE_AUDIT/personas/` which are psychological filters for evaluating creatives.

- **Use these avatars** when building offers, crafting messaging, choosing angles
- **Use Audit Personas** when evaluating/scoring existing creatives

---

## Quick Reference

| # | Name | Tagline | Primary Pain | Primary Emotion |
|---|------|---------|--------------|-----------------|
| 1 | **Marcus** | Stuck Starter | Overwhelmed, doesn't know where to begin | Overwhelm → Relief |
| 2 | **Sarah** | Side Hustle Parent | No time, needs fastest path | Exhaustion → Time Savings |

| 9 | **Carlos** | Marketplace Builder | Wants multi-vendor, thinks it costs $100K | Disbelief → "This is possible" |
