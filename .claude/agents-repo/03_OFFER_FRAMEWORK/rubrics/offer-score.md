# Offer Strength Rubric

Comprehensive scoring for offer components. Use after building offer to identify gaps.

---

## Section A: Value Equation (Core)

| Dimension | 0 | 1 | 2 | 3 |
|-----------|---|---|---|---|
| **Dream Outcome** | Vague/unclear | Understandable | Clear | Specific + measurable + timeframe |
| **Perceived Likelihood** | No proof | Weak proof | Some proof | Overwhelming proof (contexted) |
| **Time Delay** | Long, unclear | Defined but slow | Reasonable | Fast + quick wins included |
| **Effort & Sacrifice** | High effort, complex | Some effort | Manageable | Minimal (step-by-step, templates, done-for-you) |

**Value Equation Score:** /12

---

## Section B: Offer Components

| Dimension | 0 | 1 | 2 | 3 |
|-----------|---|---|---|---|
| **Transformation Statement** | Missing | Vague | Clear | Specific (avatar + outcome + mechanism + time) |
| **Unique Mechanism** | None | Generic USP | Named mechanism | Proprietary + explained + believable |
| **Offer Name** | Generic | Some elements | 3+ MAGIC elements | Full MAGIC + memorable |
| **Pricing Strategy** | Random | Cost-based | Strategically positioned | Anchored + stacked + psychological |
| **Bonuses** | None | Generic extras | Named + valued | Solves objections + exceeds core value |
| **Guarantee** | None | Weak/vague | Standard | Strong risk reversal + clear terms |
| **Urgency/Scarcity** | None | Fake/weak | Real but generic | Real + specific + explained |
| **Credibility** | None | Basic claims | Some proof | Multiple proof types, contexted |

**Offer Components Score:** /24

---

## Section C: Message Flow

| Dimension | 0 | 1 | 2 | 3 |
|-----------|---|---|---|---|
| **Hook** | Generic | Topic-relevant | Attention-getting | Pattern-breaking + self-identification |
| **Problem Agitation** | Missing | Surface level | Pain articulated | Deep pain + false solutions addressed |
| **Solution Presentation** | Features only | Benefits mentioned | Mechanism explained | Clear why it works |
| **Objection Handling** | None | Some addressed | Main objections handled | Proactive + reframes beliefs |
| **Social Proof** | None | Generic claims | Testimonials | Contexted proof (who + result + timeframe) |
| **CTA** | Missing/weak | Present | Clear action | One action + urgency + benefit |

**Message Flow Score:** /18

## Quick Checks

### Value Equation Quick Test
- [ ] Can you state the dream outcome in one specific sentence?
- [ ] Do you have 3+ pieces of contexted proof?
- [ ] Can someone see first results within days (not months)?
- [ ] Is effort minimized with templates/guides/support?

### Offer Stack Quick Test
- [ ] Would YOU buy this offer at this price?
- [ ] Is the bonus stack worth more than the core offer?
- [ ] Does the guarantee remove all risk?
- [ ] Is there a real reason to act NOW?

### Message Quick Test
- [ ] Would someone stop scrolling at the hook?
- [ ] Is the mechanism (why it works) explained?
- [ ] Are the main 2-3 objections addressed?
- [ ] Is there ONE clear CTA?
