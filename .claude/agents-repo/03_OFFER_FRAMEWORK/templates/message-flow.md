# Message Flow Template

The default persuasion sequence for ads and landing pages.

---

## The 8-Step Flow

| Step | Purpose | Content Focus |
|------|---------|---------------|
| 1. **Hook** | Stop the scroll | Pattern interrupt + self-identification |
| 2. **Jealousy/Contrast** | Create desire | Show what's possible |
| 3. **Problem** | Agitate pain | Dig into current struggle |
| 4. **Solution** | Introduce product | Present mechanism |
| 5. **Easy/Quick/Safe** | Reduce friction | Lower time delay + effort |
| 6. **Objection Handling** | Remove barriers | Address concerns |
| 7. **Social Proof** | Build belief | Testimonials + results |
| 8. **CTA** | Get action | One clean next step |

---

## Step 1: Hook

**Purpose:** Pattern interrupt + immediate self-identification

**Hook Types:**
- Question: "Still struggling with X?"
- Statement: "Most people don't know this about X..."
- Contrast: "Everyone's doing X. Here's why that's wrong."
- Story: "I used to [relatable situation]..."
- Number: "X% of [avatars] fail at Y. Here's why."
