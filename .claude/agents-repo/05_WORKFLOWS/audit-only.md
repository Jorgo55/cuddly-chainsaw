# Audit-Only Workflow

Evaluate existing creative without building offer.

**Note**: This workflow uses the 5 Audit Personas (Paula, Sophia, Devin, Ezra, Avery) - psychological filters for predicting buyer reactions. These are different from the 9 Product Avatars used in offer building.

---

## Input Required

User submits one of:
- UGC script
- Static ad copy
- Video outline
- Hook list
- Landing page

---

## Process

### 1. Run Through 5 Personas
For each persona in `04_CREATIVE_AUDIT/personas/`:
- Score fit (0-10)
- Calculate trust %
- Capture System 1 + System 2 reactions
- Write voice line

### 2. Score Conversion Radar
Using `04_CREATIVE_AUDIT/conversion-radar/scoring-anchors.md`:
- Hook (0-10)
- Emotion (0-10)
- Clarity (0-10)
- Relevance (0-10)
- Believability (0-10)
- Belief Break (0-10)

### 3. Calculate Trust Score
Using `04_CREATIVE_AUDIT/trust-scoring/trust-rubric.md`:
- Identify boosters
- Identify killers
- Calculate % (0-100)

### 4. Identify Bottleneck
Single highest-leverage issue

### 5. Deliver Verdict
Using `04_CREATIVE_AUDIT/verdicts/pass-revise-kill.md`:
- PASS / REVISE / KILL
- Reasoning

### 6. Fix Plan
1-3 changes ranked by expected lift

---

## Output

- Full persona scorecard
- Conversion radar scores
- Trust score
- Bottleneck diagnosis
- Verdict
- Fix plan (if REVISE)
