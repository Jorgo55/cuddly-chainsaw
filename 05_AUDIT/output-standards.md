# Output Standards

Exact formats for audit and offer outputs. Follow these, don't improvise.

---

## Creative Audit Output

### A) Executive Summary

```
## Executive Summary

**Verdict**: [PASS / REVISE / KILL]

**Primary Bottleneck**: [1 sentence — the single main reason it will underperform]

**Trust Score**: XX%
**Weighted Radar Average**: X.X/10

**Top Fixes (ranked by lift)**:
1. [Fix] → [expected impact]
2. [Fix] → [expected impact]
3. [Fix] → [expected impact]
```

### B) Conversion Radar

```
## Conversion Radar

| Dimension | Score | Weight | Weighted |
|-----------|-------|--------|----------|
| Hook | /10 | 20% | |
| Clarity | /10 | 20% | |
| Believability | /10 | 20% | |
| Relevance | /10 | 15% | |
| Emotion | /10 | 15% | |
| Belief Break | /10 | 10% | |
| **Weighted Average** | | | **/10** |

**Notes**:
- [2-5 bullets on what helps/hurts]
```

### C) Persona Panel (repeat for all 5, in order)

Order: Ezra → Sophia → Paula → Devin → Avery

```
## [Persona Name]

**Overall Fit**: X/10
**Trust**: XX%

**System 1 (gut)**: [1-2 lines — instant reaction]
**System 2 (logic)**: [1-2 lines — considered reaction]

**Voice Line**: "[What they'd actually say seeing this ad]"

**Radar Notes**:
- [2-4 bullets on what helps/hurts for this persona]
```

### D) Fix Plan (ranked)

```
## Fix Plan

### Fix #1: [Title]
**What to change**: [Specific action]
**Why**: [Diagnosis]
**Expected lift**: [What improves]

### Fix #2: [Title]
...

### Fix #3: [Title]
...
```

### E) Rewrite Suggestions (optional, if requested)

```
## Rewrite Suggestions

### Hook Options (3)
1. [Hook variant 1]
2. [Hook variant 2]
3. [Hook variant 3]

### Proof Inserts
- [Where]: [What proof to add]
- [Where]: [What proof to add]
```

---

## General Standards

- Commit to verdicts (no "could" hedging)
- Specificity over generality
- 1-3 fixes, not 20 tips
- Evidence or labeled assumption
- No flattery, no filler
