# Trust Score Rubric

Calculate trust percentage for any creative. Used alongside Conversion Radar.

---

## Trust Builders (Add Points)

| Signal | Points | Notes |
|--------|--------|-------|
| Specific numbers/metrics | +10 | "247 founders" > "hundreds" |
| Named mechanism | +10 | Proprietary system/process |
| Contexted testimonial | +10 | Who + result + timeframe |
| Time constraint on claim | +5 | "in 90 days" vs "quickly" |
| "Who it's NOT for" | +5 | Exclusion = confidence |
| Third-party validation | +10 | Awards, press, platform reviews |
| Founder/face visible | +5 | Personal accountability |
| Demo/screenshot | +10 | Show, don't tell |
| Money-back / guarantee | +5 | Risk reversal |
| Limitations acknowledged | +5 | Honesty = trust |

---

## Trust Breakers (Subtract Points)

| Signal | Points | Notes |
|--------|--------|-------|
| Vague superlatives ("best," "amazing") | -10 | Empty claims |
| No proof for big claim | -15 | Claim > evidence |
| Hype tone / exclamation marks | -5 | Salesy feel |
| Stock imagery / generic visuals | -5 | Feels fake |
| No face / anonymous | -5 | Who's behind this? |
| "Guaranteed results" without terms | -10 | Compliance risk |
| Too many promises | -5 | Cognitive overload |
| No specificity on audience | -5 | Generic = for no one |

---

## Scoring

1. Start at 50% (neutral baseline)
2. Add trust builders
3. Subtract trust breakers
4. Cap at 100%, floor at 0%

### Interpretation

| Trust Score | Meaning |
|-------------|---------|
| 80-100% | Strong — creative earns belief |
| 70-79% | Good — minor trust gaps |
| 55-69% | Weak — needs more proof/specificity |
| 45-54% | Poor — significant trust issues |
| < 45% | AUTO-KILL — doesn't earn belief |
